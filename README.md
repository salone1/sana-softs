# SANA Softs - Professional Software Company Website

A modern, responsive, and advanced company website showcasing SANA Softs' apps, services, and vision. Built with React + Vite + Tailwind CSS for speed, scalability, and a polished UI.

## 🌟 Features

### Pages & Functionalities

1. **Home Page** - Hero section with mission statement, company introduction, featured apps, and services overview
2. **Apps Page** - Showcase of all developed applications with grid layout and download links
3. **App Detail Page** - Full app descriptions with features, screenshots, use cases, changelog, and download buttons (APK & Google Play)
4. **Contact Page** - Contact form, social links (Telegram, Instagram), and FAQ section
5. **About Page** - Company history, vision, mission, values, team information, and core principles
6. **Services Page** - Detailed list of services with features, pricing, process, and why choose us section
7. **Careers Page** - Job opportunities, company culture, hiring process, and benefits
8. **Blog Page** - Articles on software trends, tutorials, company updates with categorization and filtering
9. **Privacy Policy Page** - GDPR-compliant data handling transparency
10. **Terms & Conditions Page** - Legal terms for using apps and services

## 📦 Installation

1. Clone the repository:
```bash
git clone https://github.com/yourusername/sana-softs.git
cd sana-softs
```

2. Install dependencies:
```bash
npm install
```

3. Start the development server:
```bash
npm run dev
```

4. Build for production:
```bash
npm run build
```

5. Preview the build:
```bash
npm run preview
```

## 🚀 Deployment

### Deploy on Cloudflare Pages

1. Push your code to GitHub
2. Connect your GitHub repository to Cloudflare Pages
3. Set the build command: `npm run build`
4. Set the build output directory: `dist`
5. Deploy!

### Environment Variables

Create a `.env` file for any environment-specific variables:

```
VITE_API_URL=https://api.sanasofts.com
```

## 📝 Customization

### Update Company Information

- Edit contact details in `src/pages/Contact.jsx`
- Update social media links in `src/components/Footer.jsx`
- Modify app information in `src/pages/Apps.jsx` and `src/pages/AppDetail.jsx`
- Update founder information in `src/pages/Home.jsx` and `src/pages/About.jsx`

### Add New Pages

1. Create a new component in `src/pages/`
2. Import and add the route in `src/App.jsx`
3. Add navigation link in `src/components/Navbar.jsx`

### Customize Styling

Edit `src/index.css` and `tailwind.config.js` to customize colors, fonts, and other design elements.
